A bit about me

I’m a software developer from London.

From my time at university, I acquired a particular interest in web applications due to their versatile nature, especially with the growth of Progressive Web Apps (PWA). I primarily have experience using Ruby on Rails however, I have begun an exploration into other web application platforms with the use of Angular to create this PWA.

I love programming because I find it incredibly rewarding when you’re able to make someone else’s life a little bit easier by doing something you enjoy. Also, programming is just fun! It’s challenging and therefore really satisfying when you solve problems and see something develop that you've built from the ground up.

Now as a graduate, I look forward to utilising, improving and adding to my current skill set in a fun, collaborative and challenging environment under the mentorship of more experienced teammates.
